Currency Pipe Example

Formatted amount using currency pipe: {{ amount | currency }}

Amount with USD currency symbol: {{ amount | currency:'USD' }}

Amount with currency code and symbol: {{ amount | currency:'EUR':'symbol' }}

Amount with currency code and display style: {{ amount | currency:'GBP':'code':'4.2-2' }}


Decimal Pipe Example

Formatted number with default precision: {{ piValue | number }}

Number with specific precision (2 digits): {{ num1 | number:'1.2' }}

Number with zero padding to the left (3 digits in total): {{ piValue | number:'1.3' }}


Percentage Pipe Example

Success Rate: {{ successRate | percent }}

Improved Success Rate: {{ successRate + 0.15 | percent }}
